# HTML & CSS Practice Portfolio

This repository contains a collection of 10 tasks focusing on semantic HTML, CSS styling, responsive design without JavaScript, and advanced CSS techniques. Each task demonstrates practical implementations of modern web design principles.

## Tasks Overview

### 1. Static Webpage Layout ([Task1](./Task1))
* **Objective:** Create a single-page website using semantic HTML elements (`<header>`, `<nav>`, `<main>`, `<footer>`).
* **Highlights:** 
  * Basic CSS typography and color styling.
  * Margin and padding for clear structure.
  * Simple responsive design with media queries.

### 2. CSS Card Component with Hover Effects ([Task2](./Task2))
* **Objective:** Build a stylized card component with an image, title, and description.
* **Highlights:**
  * Borders, shadows, and padding.
  * Smooth CSS transitions on hover (e.g., background color change, slight scaling).

### 3. Responsive Grid Layout ([Task3](./Task3))
* **Objective:** Display multiple items in a structured grid layout.
* **Highlights:**
  * CSS Grid used for a 3-column desktop layout.
  * Responsive stacking into a single column on mobile.
  * Consistent spacing and alignment.

### 4. Pure CSS Dropdown Menu ([Task4](./Task4))
* **Objective:** Design a navigation bar with a hover-triggered dropdown menu.
* **Highlights:**
  * Nested `<ul>` for menu and submenu.
  * `:hover` pseudo-class and transitions for smooth reveal.
  * Responsive fallback formatting.

### 5. Modal Popup (Pure CSS) ([Task5](./Task5))
* **Objective:** Open and close a modal window without any JavaScript.
* **Highlights:**
  * Utilizes `:target` or hidden checkbox (`:checked`) hack.
  * Centered modal with an overlay backdrop.
  * Smooth appearance/disappearance transitions.

### 6. Tabbed Content Interface ([Task6](./Task6))
* **Objective:** Build a tabbed UI where clicking tabs displays different content sections.
* **Highlights:**
  * Hidden radio buttons/checkboxes with labels.
  * `:checked` state to toggle visibility of content.
  * Smooth content switching transitions.

### 7. Pure CSS Carousel/Slider ([Task7](./Task7))
* **Objective:** Develop an image carousel that slides manually or automatically.
* **Highlights:**
  * `@keyframes` for auto-sliding animations.
  * `:checked` or `:target` pseudo-classes for manual slide selection.
  * Responsive and visually engaging design.

### 8. CSS Accordion Component ([Task8](./Task8))
* **Objective:** Create an accordion with expanding and collapsing content sections.
* **Highlights:**
  * Checkbox hack or `:target` pseudo-class to control open/closed states.
  * Smooth height expansion/collapse transitions.
  * Support for single or multiple open sections simultaneously.

### 9. Complex Responsive Layout (Grid & Flexbox) ([Task9](./Task9))
* **Objective:** Construct a sophisticated layout combining modern CSS layout modules.
* **Highlights:**
  * CSS Grid for main layout structure; Flexbox for inner alignment.
  * Overlapping elements with `position` and `z-index`.
  * Dynamic reordering on resize.
  * Sticky header or sidebar during scroll.

### 10. Interactive Multi-Page Website Simulator ([Task10](./Task10))
* **Objective:** Simulate a fully functional, multi-page website experience purely via CSS.
* **Highlights:**
  * Uses the `:target` pseudo-class to show/hide "pages" dynamically.
  * CSS animations for page transitions (sliding/fading).
  * Accessible responsive navigation.
  * Combines Flexbox, Grid, and pseudo-classes to manage state.
